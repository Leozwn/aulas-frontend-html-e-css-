O poder das variáveis

As variáveis (ou Custom Properties) tornam o código CSS mais fácil de manter e orhanizar, pricnipalmente quando se precisa mudar aspectos visuais de acordo com alguma preferência ou intenção do usuário (exemplo:mudança de paleta de cores ou tema visual).

Exemplo: se quisermos mudar a cor do link abaixo e do título, basta alterar apenas a varável que define a cor desses elementos
