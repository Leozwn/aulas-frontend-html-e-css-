Media Queries - Mobile First

Conceito

Media Queries são regras especiais em que criamos alterações na página de acordo com determinadas condições (exemplo: tamanho da tela).

Essas alterações ocorrem com a configuração de breakpoints.

As media queries são aplicadas de acordo com a estratégia usada para design responsivo, podendo ser:

  • Mobile First: do pequeno para o grande
  • Desktop First: do grnade para o pequeno
